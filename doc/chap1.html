1.1-1 LeftSeries
‣ LeftSeries( A )( attribute )

Returns: a list of left ideals

The left series of a skew brace A is defined recursively as A^1=A and A^{n+1}=A*A^n for n\geq1, where a*b=\lambda_a(b)-b

 gap> br := SmallSkewBrace(8,20);
 <skew brace of size 8>
 gap> LeftSeries(br);
 [ <skew brace of size 8>, <brace of size 2>, <brace of size 1> ]

1.1-2 SmoktunowiczSeries
‣ SmoktunowiczSeries( A )( attribute )

Returns: a list of ideals

The Smoktunowicz's series of a skew brace A is defined recursively as A^{[1]}=A and A^{[n+1]}=\sum_{i=1}^n A^{[i]}*A^{[n+1-i]} for n\geq1, where a*b=\lambda_a(b)-b.

 gap> br := SmallSkewBrace(10,2);
 <skew brace of size 10>
 gap> SmoktunowiczSeries(br);
 [ <skew brace of size 10>, <brace of size 1> ]

1.4-1 IsTwoSided
‣ IsTwoSided( arg )( property )

Returns: true if the skew brace is two sided, false otherwise

A skew brace A is said to be two-sided if (a+b)\circ c=a\circ c-c+b\circ c holds for all a,b,c\in A.

 gap> IsTwoSided(SmallSkewBrace(8,2));
 false
 gap> IsTwoSided(SmallSkewBrace(8,4));
 true
